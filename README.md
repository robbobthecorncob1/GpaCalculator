# GPA Calculator Frontend

The frontend companion to ProfileCore. This is a modern, responsive web application built with React, Vite, and TypeScript. It currently features a fully functional GPA Calculator that leverages the ProfileCore API for precise calculations.

## Prerequisites to run locally

* [Node.js](https://nodejs.org/en/download) (Latest LTS recommended)

* ProfileCore API running locally (see [ProfileCore](https://github.com/robbobthecorncob1/ProfileCore) for setup)
